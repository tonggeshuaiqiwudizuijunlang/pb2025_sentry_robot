## 弹道可视化调试指令

### 编译弹道可视化模块
```bash
# 单独编译 armor_tracker（包含弹道可视化）
colcon build --packages-select armor_tracker

# 编译后 source
source install/setup.bash
```

### 弹道参数配置
```bash
# 运行时动态修改弹速参数
ros2 param set /armor_tracker trajectory.bullet_speed 25.0

# 查看当前弹道参数
ros2 param get /armor_tracker trajectory.bullet_speed
ros2 param get /armor_tracker trajectory.compensator_type
ros2 param get /armor_tracker trajectory.gravity
ros2 param get /armor_tracker trajectory.resistance

# 切换弹道模型（ideal=理想弹道，resistance=考虑空气阻力）
ros2 param set /armor_tracker trajectory.compensator_type "resistance"
```

### RViz 查看弹道轨迹
```bash
# 启动 RViz
rviz2

# 在 RViz 中添加 MarkerArray，订阅话题：
# /tracker/marker

# 弹道轨迹颜色说明：
# - 粉色：正在跟踪目标
# - 绿色：瞄准稳定（yaw/pitch 偏差 < 0.1 rad）
```

### 话题调试
```bash
# 查看 tracker 发布的所有话题
ros2 topic list | grep tracker

# 查看目标信息
ros2 topic echo /tracker/target

# 查看 marker 数据
ros2 topic echo /tracker/marker

# 查看话题频率
ros2 topic hz /tracker/target
ros2 topic hz /tracker/marker
```

### 常见问题排查
```bash
# 检查 TF 树是否正常（弹道可视化需要 gimbal_link 坐标系）
ros2 run tf2_tools view_frames

# 查看 TF 变换
ros2 run tf2_ros tf2_echo odom gimbal_link

# 检查节点是否正常运行
ros2 node list | grep tracker
ros2 node info /armor_tracker

# 查看节点日志
ros2 run rqt_console rqt_console
```

### 弹道参数参考值
```yaml
# 17mm 弹丸（步兵/英雄）
trajectory:
  compensator_type: "ideal"
  bullet_speed: 15.0      # 15-30 m/s
  gravity: 9.8
  resistance: 0.01

# 42mm 弹丸（英雄大弹丸）
trajectory:
  compensator_type: "resistance"
  bullet_speed: 16.0
  gravity: 9.8
  resistance: 0.001
```

### 录制调试数据
```bash
# 录制相关话题
ros2 bag record /tracker/target /tracker/marker /detector/armors -o trajectory_debug

# 回放
ros2 bag play trajectory_debug
```

## 自瞄系统调试指令

### 启动自瞄系统
```bash
# 完整启动自瞄
ros2 launch rm_vision_bringup vision_bringup.launch.py

# 单独启动相机
ros2 launch hik_camera hik_camera.launch.py

# 单独启动检测器
ros2 launch armor_detector armor_detector.launch.py

# 单独启动追踪器
ros2 launch armor_tracker armor_tracker.launch.py
```

### 相机调试
```bash
# 查看相机图像
ros2 run rqt_image_view rqt_image_view

# 相机话题
ros2 topic echo /camera/image_raw --no-arr
ros2 topic hz /camera/image_raw

# 查看相机信息
ros2 topic echo /camera/camera_info

# 动态调整相机参数（曝光、增益等）
ros2 run rqt_reconfigure rqt_reconfigure
```

### 装甲板检测调试
```bash
# 查看检测结果话题
ros2 topic echo /detector/armors

# 查看检测结果图像
ros2 topic list | grep detector
# 通常是 /detector/result_img

# 检测频率
ros2 topic hz /detector/armors

# 动态调整检测参数
ros2 run rqt_reconfigure rqt_reconfigure
# 可调参数：二值化阈值、灯条筛选参数、装甲板筛选参数等
```

### 追踪器调试
```bash
# 查看追踪目标
ros2 topic echo /tracker/target

# 查看追踪状态（0=LOST, 1=DETECTING, 2=TRACKING, 3=TEMP_LOST）
ros2 topic echo /tracker/tracking_state

# 查看追踪信息（位置差、yaw差等）
ros2 topic echo /tracker/info

# 查看陀螺状态
ros2 topic echo /tracker/top

# 重置追踪器
ros2 service call /tracker/reset std_srvs/srv/Trigger

# 切换目标
ros2 service call /tracker/change std_srvs/srv/Trigger
```

### EKF 参数调试
```bash
# 查看 EKF 参数
ros2 param get /armor_tracker ekf.sigma2_q_xyz_max
ros2 param get /armor_tracker ekf.sigma2_q_xyz_min
ros2 param get /armor_tracker ekf.sigma2_q_yaw_max
ros2 param get /armor_tracker ekf.sigma2_q_yaw_min
ros2 param get /armor_tracker ekf.sigma2_q_r
ros2 param get /armor_tracker ekf.r_xyz_factor
ros2 param get /armor_tracker ekf.r_yaw

# 动态修改 EKF 参数
ros2 param set /armor_tracker ekf.sigma2_q_xyz_max 0.2
ros2 param set /armor_tracker ekf.r_xyz_factor 0.08
```

### 串口通信调试
```bash
# 启动串口
ros2 launch standard_robot_pp_ros2 standard_robot_pp_ros2.launch.py

# 查看串口接收数据（云台姿态等）
ros2 topic echo /serial/receive

# 查看串口发送数据（控制指令）
ros2 topic echo /serial/send

# 检查串口设备
ls -l /dev/ttyUSB*
ls -l /dev/ttyACM*

# 串口权限
sudo chmod 777 /dev/ttyUSB0
sudo usermod -aG dialout $USER
```

### TF 坐标系调试
```bash
# 查看 TF 树
ros2 run tf2_tools view_frames
# 生成 frames.pdf

# 实时查看 TF
ros2 run tf2_ros tf2_echo odom camera_optical_frame
ros2 run tf2_ros tf2_echo odom gimbal_link
ros2 run tf2_ros tf2_echo camera_optical_frame camera_link

# TF 监控
ros2 run tf2_ros tf2_monitor
```

### RViz 可视化配置
```bash
# 启动 RViz
rviz2

# 需要添加的显示项：
# 1. TF - 查看坐标系
# 2. MarkerArray - 话题 /tracker/marker（装甲板、速度、弹道）
# 3. Image - 话题 /detector/result_img（检测结果）
# 4. Camera - 话题 /camera/image_raw

# 固定坐标系设置为 odom
```

### 性能监控
```bash
# 查看节点 CPU/内存占用
htop

# 查看话题带宽
ros2 topic bw /camera/image_raw
ros2 topic bw /detector/armors

# 查看话题延迟
ros2 topic delay /detector/armors
ros2 topic delay /tracker/target

# 查看所有节点
ros2 node list
```

### 日志与录包
```bash
# 查看日志
ros2 run rqt_console rqt_console

# 设置日志级别
ros2 service call /armor_detector/set_logger_level rcl_interfaces/srv/SetLoggerLevel "{logger_name: 'armor_detector', level: 10}"
# level: 10=DEBUG, 20=INFO, 30=WARN, 40=ERROR

# 录制自瞄相关话题
ros2 bag record \
  /camera/image_raw \
  /camera/camera_info \
  /detector/armors \
  /tracker/target \
  /tracker/marker \
  /tf \
  /tf_static \
  -o auto_aim_debug

# 回放
ros2 bag play auto_aim_debug --clock
```

### 常用调试组合
```bash
# 终端1：启动自瞄
ros2 launch rm_vision_bringup vision_bringup.launch.py

# 终端2：查看图像
ros2 run rqt_image_view rqt_image_view

# 终端3：动态调参
ros2 run rqt_reconfigure rqt_reconfigure

# 终端4：RViz 可视化
rviz2

# 终端5：监控话题
watch -n 0.5 'ros2 topic hz /tracker/target'
```

### 问题排查清单
```bash
# 1. 相机无图像
ls /dev/video*                    # 检查相机设备
ros2 topic hz /camera/image_raw   # 检查话题频率

# 2. 检测不到装甲板
ros2 run rqt_reconfigure rqt_reconfigure  # 调整二值化阈值
ros2 topic echo /detector/armors          # 查看检测结果

# 3. 追踪不稳定
ros2 topic echo /tracker/info             # 查看位置差/yaw差
ros2 param set /armor_tracker ekf.sigma2_q_xyz_max 0.05  # 降低过程噪声

# 4. TF 报错
ros2 run tf2_tools view_frames            # 检查 TF 树完整性
ros2 run tf2_ros tf2_monitor              # 检查 TF 发布频率

# 5. 串口通信失败
ls -l /dev/ttyUSB*                        # 检查设备
sudo chmod 777 /dev/ttyUSB0               # 权限
ros2 topic echo /serial/receive           # 检查接收
```
